# Mobile Security Labs

This repository contains mobile application security testing notes, lab-based findings, and vulnerability assessment writeups based on intentionally vulnerable applications.

The purpose of this repository is to document mobile security testing methodology, risk analysis, and remediation recommendations in a professional and portfolio-safe manner.

## Focus Areas

- Android Application Security
- OWASP Mobile Top 10
- Static Analysis
- Dynamic Analysis
- Insecure Data Storage
- Exported Components
- Hardcoded Secrets
- Weak Authentication

## Tools Used

- MobSF
- JADX
- adb
- Android Studio
- Burp Suite
- OWASP ZAP
